## Getting Started

### Prerequisites

- A modern web browser

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Sirwagya/weather_app.git
   ```

2. Navigate to the project folder:

   ```bash
   cd weather_app
   ```

3. Open `index.html` in your browser or use a live server.
